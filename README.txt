SITE LIG AUTOMAÇÃO - VERSÃO DO ZERO

Esta versão foi criada do zero usando apenas a paleta visual da imagem enviada como referência:
- laranja/vermelho como cor principal
- grafite/preto como base de contraste
- cinza/prata como cor secundária
- branco/cinza claro para áreas limpas

Arquivos principais:
- index.html: estrutura do site
- styles.css: identidade visual e responsividade
- script.js: menu mobile, animações e formulário via WhatsApp
- assets/img/logo-lig-automacao.png: logo usada no topo e rodapé
- assets/img/hero-automacao.svg: ilustração vetorial criada para o site
- assets/img/favicon.png: ícone do navegador
